C++ Ch12 动态内存与智能指针

发表于 2021-04-01 分类于 c++

静态内存：保存局部static变量、类static变量和在函数体外定义的变量
栈内存：保存定义在函数体内的非static变量
分配在静态内存和栈内存中的对象由编译器自动创建或销毁；栈对象仅在程序块运行时存在；static在程序结束时销毁

此外，每个程序还拥有一个内存池（堆内存）用于动态分配的对象；这类对象的生存周期由程序控制，即需要代码显示销毁
